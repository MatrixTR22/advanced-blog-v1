# Advanced Blog v1

Bu proje, modern ve interaktif bir blog/haber sitesi örneğidir.  
Temel özellikler:

- Dinamik haberler ve yazılar
- Kategori filtreleme
- Arama özelliği
- Light/Dark mod değişimi
- Responsive tasarım
- Hover ve fade-in animasyonları
- Post detay sayfası ve yazar bilgisi

## Kullanım

1. Tüm dosyaları bilgisayarınıza indirin
2. `index.html` dosyasını tarayıcıda açın
3. JS ile dinamik olarak içerik görüntülenir
